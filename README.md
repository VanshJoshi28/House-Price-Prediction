# House Price Prediction Using Machine Learning

## Project Overview

This project predicts house prices using various housing features such as area, number of bedrooms, bathrooms, parking spaces, and furnishing status. The model is built using Linear Regression and demonstrates a complete machine learning workflow from data preprocessing to prediction.

---

## Features

- Data Cleaning and Preprocessing
- Feature Encoding
- Exploratory Data Analysis (EDA)
- Data Visualization
- Linear Regression Model Training
- Model Evaluation
- House Price Prediction

---

## Dataset

The project uses a housing dataset containing features such as:

- Area
- Bedrooms
- Bathrooms
- Stories
- Parking
- Main Road Access
- Guest Room
- Basement
- Hot Water Heating
- Air Conditioning
- Preferred Area
- Furnishing Status
- Price

## Machine Learning Workflow

### 1. Data Loading
The housing dataset is loaded using Pandas.

### 2. Data Preprocessing
- Converted Yes/No categorical values into numerical values.
- Applied One-Hot Encoding to furnishing status.

### 3. Exploratory Data Analysis
- Price Distribution
- Area vs Price Analysis
- Correlation Heatmap

### 4. Model Training
A Linear Regression model was trained using Scikit-Learn.

### 5. Model Evaluation
The model performance was evaluated using:

- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- R² Score

### 6. Prediction
The trained model predicts house prices based on housing features.
